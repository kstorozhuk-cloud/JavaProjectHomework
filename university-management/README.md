# University Management System (CLI)

Консольна система управління університетом на Java: студенти, викладачі, курси, зарахування, оцінки, GPA, звіти та пошук.

Фінальний проєкт з курсу Java.

## Можливості

- **Студенти** — CRUD-операції, зміна статусу (`ACTIVE`, `ON_LEAVE`, `EXPELLED`, `GRADUATED`), фільтри за статусом і роком навчання, сортування за ПІБ (bubble sort).
- **Викладачі** — CRUD-операції, посади через enum `TeacherPosition` (`ASSISTANT`, `LECTURER`, `DOCENT`, `PROFESSOR`).
- **Курси** — CRUD-операції, фільтри за викладачем та кількістю кредитів.
- **Зарахування** — зарахування студента на курс у семестрі (початково оцінка `NA`, оплата `false`), виставлення оцінок (enum `Grade`: `A`, `B`, `C`, `D`, `F`, `NA`), позначення оплати (інтерфейс `Payable`), перегляд зарахувань студента з GPA, транскрипт у консолі.
- **Звіти / Пошук** — пошук студентів за частиною ПІБ чи email, список неоплачених зарахувань, середній бал по курсу та за семестр, топ-N студентів за GPA.

## Технічні деталі

- Дані зберігаються в пам'яті програми у **масивах** (без колекцій), з ручним розширенням місткості.
- ООП: наслідування (`Person` → `Student`, `Teacher`), інтерфейс `Payable`, enum, інкапсуляція (приватні поля + валідація у сеттерах), `equals`/`hashCode`/`toString` у сутностях.
- GPA студента — середній бал, **зважений за кредитами** курсів; оцінки `NA` не враховуються.
- Сортування — простий **bubble sort** (`GPAUtils`).
- Некоректний ввід (email без `@`, рік навчання поза межами 1–6, нечислове значення, неіснуючий id, повторне зарахування) → `IllegalArgumentException` зі зрозумілим повідомленням. Усі винятки перехоплюються в меню — програма не завершується аварійно.
- При видаленні студента або курсу видаляються і пов'язані зарахування; викладача, за яким закріплені курси, видалити не можна.

## Структура проєкту

```
src/
 └── university/
      ├── Main.java                 // головний клас з меню
      ├── entities/
      │    ├── Person.java          // базовий клас (ПІБ, email)
      │    ├── Student.java         // рік навчання, статус
      │    ├── Teacher.java         // посада
      │    ├── Course.java          // назва, кредити, викладач
      │    └── Enrollment.java      // студент + курс + семестр, оцінка, оплата
      ├── enums/
      │    ├── Grade.java           // A, B, C, D, F, NA (+ бали для GPA)
      │    ├── StudentStatus.java
      │    └── TeacherPosition.java
      ├── interfaces/
      │    └── Payable.java         // isPaid() / markPaid()
      ├── services/
      │    ├── StudentService.java
      │    ├── TeacherService.java
      │    ├── CourseService.java
      │    └── EnrollmentService.java
      └── util/
           └── GPAUtils.java        // GPA, bubble sort
```

## Вимоги

- Java 11 або новіша (рекомендовано 17+).

## Запуск

### Linux / macOS

```bash
javac -encoding UTF-8 -d out $(find src -name "*.java")
java -cp out university.Main
```

### Windows (cmd)

```cmd
dir /s /b src\*.java > sources.txt
javac -encoding UTF-8 -d out @sources.txt
java -cp out university.Main
```

> Якщо кирилиця у консолі Windows відображається некоректно, виконай `chcp 65001` перед запуском або запускай з IDE.

### IntelliJ IDEA

Відкрий папку проєкту, познач `src` як Sources Root (зазвичай визначається автоматично) і запусти клас `university.Main`.
